📖 사용 설명서

시작하기

1️⃣ 좌측 빌더에서 옵션들을 선택하세요
2️⃣ 과목, 학년, 결과물 형태는 필수입니다
3️⃣ 웹 PPT나 구글 슬라이드 선택 시 수업 차시가 나타납니다
4️⃣ 우측 프리뷰에서 선택사항을 확인하세요
5️⃣ 구체적인 주제를 입력하고 생성 버튼을 누르세요
6️⃣ 프롬프트가 자동으로 복사되면 AI에 붙여넣기 하세요

수업 차시 및 구성

🔵 1차시 (45분): 도입 → 전개 → 정리
🟢 2차시 (90분): 1차시 + 1차시 (복습 후 심화)
🟠 3차시 (135분): 1→2→3차시 (단계적 심화)
🔴 4차시 (180분): 1~4차시 (전문화 과정)

수업 흐름:
• 도입(5~10분): 동기유발, 이전 차시 복습, 학습목표 제시
• 전개(25~30분): 강의, 학습활동, 토론, 실습 (차시별 심화)
• 정리(5~10분): 핵심 내용 정리, 차시 예고, 과제 안내

유튜브 활용 방식

📝 내용 요약: 영상의 주요 내용을 정리하여 슬라이드에 통합
🔗 영상 주소 첨부: QR코드나 링크로 학생들이 수업 후 시청
🖼️ 시각화 강화: 영상의 시각 요소를 분석하여 슬라이드에 반영
💬 토론자료화: 영상의 내용으로 토론 질문과 활동자료 생성

⚠️ 주의사항
- 과목, 결과물 형태, 주제는 필수입니다
- 웹 PPT/구글 슬라이드 선택 시 수업 차시를 반드시 선택하세요
- 유튜브 URL은 youtube.com 또는 youtu.be 이어야 합니다
- 주제는 구체적일수록 좋은 결과를 얻을 수 있습니다
